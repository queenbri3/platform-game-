# Platform Game Assignment 
## Project description
> Mobile compatible platform game (using )

This project is a 2D platform game inspired by the classic Mario game but with a more modern design, characters, different environments, mechanics and art.
The goal of the game is for the player to find their way through multiple levels, avoid enemies, collect coins and reach the end of the level. 
The project is developed using the Agile Scrum methodology with iterative development cycles.
